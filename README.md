## Pasos de Ejecución
Para ejecutar se necesita maven y java 17

en la carpeta de proyecto
```bash
cd sistema-calificaciones/
```
despues ejecutar 
```bash
mvn clean install
```
para generar el .jar 
Despues ejecutar
```bash
mvn javafx:run
```
